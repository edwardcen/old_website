Pong
Pong is a two player game I created that centers around bouncing a ball using paddles that are user controlled to prevent the ball from going past the screen. The game keeps track of the two players scores and resets after each goal. The game is designed to be interactive, with javascript tracking of mouse location to enable buttons on HTML canvas. To introduce an element of randomness, the ball's intial direction is highly variant, but always at the same speed.
Stock Prices Web Scraper
I designed an interpreter using a read_eval_print loop and string manipulation that can continuously read in user input about the stocks they wish to track and which exact data points they are interested in. Then I used the standard requests library in Python to pull real-time data from the Yahoo Finance API and parse it into a Pandas dataframe that can then be displayed. The interpreter also keeps track of the history of searches such that the user can easily switch between different stocks without having to reload information.
Food Atlas Data Analysis
Using the Foot Atlas open-source dataset, I developed qualitiative and categorical plots that relate various factors including demographics, access to food, median income and more to overall health of each county, measured by the adult obesity proportion. I also created a linear regression model on obesity rate and a logistic regression model on classifying states with up to 11% improved accuracy than null methods. All work done using Jupyter Notebook and with standard scientific python libraries. Notebooks, datasets used, and accompanying writeup included in Github repository.
